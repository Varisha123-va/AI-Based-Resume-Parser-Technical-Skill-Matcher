# 🤖 AI Based Resume Screening and Job Recommendation System

## 📌 Project Overview
This project is an AI-based web application that analyzes resumes and recommends suitable job roles.  
It helps users understand their resume strength by calculating a match score and identifying missing skills.

---

## 🚀 Features
- 📄 Resume Upload (PDF)
- 📊 Resume Score Calculation
- 💼 Job Recommendations
- ❗ Missing Skills Detection
- 📈 JD (Job Description) Match Score
- 💡 Suggestions for Resume Improvement
- 📥 Downloadable Report

---

## 🛠️ Technologies Used

### 🔹 Frontend
- HTML  
- CSS  
- JavaScript  

### 🔹 Backend
- Python

### 🔹 Machine Learning
- TF-IDF Vectorizer  
- Cosine Similarity  

### 🔹 Libraries
- Pandas  
- PDFPlumber  
- ReportLab  

---

## ⚙️ How It Works
1. User uploads resume PDF  
2. System extracts text from resume  
3. Skills are analyzed using AI techniques  
4. Resume is compared with job dataset  
5. System shows:
   - Resume score  
   - Job matches  
   - Missing skills  
   - Suggestions  

---

## 📂 Dataset
- jobs.csv (contains job roles and descriptions)
